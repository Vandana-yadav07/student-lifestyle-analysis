# Student Lifestyle Data Analysis
This project analyzes how lifestyle factors affect academic performance.

Tools Used:
- Python
- Pandas
- Seaborn
- Matplotlib

Key Analysis:
- Sleep vs CGPA
- Screen Time vs Stress
- Lifestyle correlations

## Key Insight
Students with higher screen time tended to report higher stress levels, while moderate sleep hours showed better academic performance.

## Dataset
Data was collected using a custom Google Form survey with ~60 student responses.
